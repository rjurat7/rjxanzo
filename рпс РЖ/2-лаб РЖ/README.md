# Лабораторная работа №2 — Шейкерная сортировка

## Цель
Разработать консольное приложение для сортировки массива целых чисел (шейкерная сортировка) и оценить сложность алгоритма.

## Структура проекта
- `main.cpp` — консольное приложение (ввод/генерация/файл → сортировка → вывод → сохранение)
- `shaker_sort.h/.cpp` — модуль сортировки (без использования `std::sort`)
- `rps2_tests/` — unit-тесты (запускаются отдельно от основного приложения)
- `report/REPORT.md` — отчёт (блок‑схема, расчёт сложности, результаты тестов)

## Как запустить приложение
Открой `rps2.sln` в Visual Studio → выбери конфигурацию `Debug` и платформу `x64` или `Win32` → **Локальный отладчик Windows**.

## Как запустить тесты
Открой `rps2.sln` → **Тест → Обозреватель тестов → Запуск всех тестов**.
