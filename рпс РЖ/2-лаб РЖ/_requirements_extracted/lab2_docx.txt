Лабораторная работа № 2
Тема: «Изучение реализации и расчёта сложности алгоритмов сортировки»
Цель работы: разработать консольное приложения для сортировки целых чисел согласно варианту и оценка сложности полученного алгоритма.
Задачи:
Описать алгоритм сортировки (согласно варианту) в виде блок-схемы.
Рассчитать лучшую и худшую временную сложность реализованного алгоритма
В репозитории из первой работы создать отдельную ветку под проект консольного приложения.
Запрограммировать описанный алгоритм сортировки в виде отдельного модуля (не используя штатные функции сортировки).
Протестировать запрограммированный алгоритм. Необходимо реализовать unit-тесты для модуля сортировки. Тесты должны запускаться отдельно от основного приложения.
Написать отчёт по работе.
Основные требования к функционалу консольного приложения:
Ввод массива пользователем. Пользователь должен выбрать один из вариантов:
Ввод массива с клавиатуры;
Генерация массива случайных целых чисел;
Загрузка массива из файла.
Отсортировать массив.
Вывод изначального и отсортированного массива на экран. Предложить пользователю выгрузить изначальный и отсортированный массива в файл.
Код должен быть написан в одной стилистике (соблюдаться правила именования переменных, функций и т.д.). Преподаватель может уточнить, какое соглашение о написании кода вы использовали.
Код должен быть прокомментирован или понятен без комментариев.
Название коммитов должно быть осмысленным.
Структура отчёта:
Цель работы.
Задание.
Алгоритм сортировки в виде блок-схемы.
Расчёт лучшей и худшей сложности алгоритма. Должно быть подробное описание расчёта, а не просто ответ.
Тестирование программы (результаты выполнения unit-тестов).
